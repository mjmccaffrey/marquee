**Marquee semi-retro lighted sign probject**

I have wanted to make this project for a while, and my son's graduation from University Alabama was the excuse I needed.<br/><br/>
As a bonus, I got to use my table saw for the first time!<br/><br/>
While the relay board and Raspberry Pi are relatively modern, the incandescent bulbs and ceramic lamp holders give it a retro feel, as do the mechanical relays, whose mechanical clicks resonate in the wooden cabinet.<br/><br/>

I have added a button to the back to allow the user to change modes, along with an RS-232 port for Raspberry Pi console access.
